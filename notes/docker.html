Docker 📦 Image Command: List images - docker images Pull image - docker pull Remove image - docker rmi Build image - docker build -t . 🧱 Container Commands: List running containers - docker ps List all containers (including stopped) - docker ps -a Run a container - docker run Run in background (detached) - docker run -d Run with port mapping - docker run -p 8080:80 Name a container - docker run --name mycontainer Start a container - docker start Stop a container - docker stop Restart a container - docker restart Remove container - docker rm Run interactive shell - docker run -it bash Exec command in running container - docker exec -it bash 🔍 Inspect & Logs : Inspect container details - docker inspect View container logs - docker logs Follow real-time logs - docker logs -f 📂Volumes & Data : List volumes - docker volume ls Create a volume - docker volume create Mount volume to container - docker run -v :/path/in/container 🧹 Cleanup Commands : Remove all stopped containers - docker container prune Remove all unused images - docker image prune Remove all unused volumes - docker volume prune Remove everything unused - docker system prune 🧰 Docker Compose Start containers - docker-compose up Start in background - docker-compose up -d Stop containers - docker-compose down Rebuild services - docker-compose up --build
